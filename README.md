## Try It

### Without cjdns
```
npm install
node server.js
```
Then, point your browser to localhost:3001

### With cjdns

Please follow the instructions in [cjdns] to set up cjdns on your computer.

[cjdns]: https://github.com/cjdelisle/cjdns

Then host the chat, and enjoy!
